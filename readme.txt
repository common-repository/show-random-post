Show_Random_Post allows users generate a Widget that automatically shows posts by category on a timed loop automatically.

== Description ==

Show_Random_Post allows users to show posts by category on a widget on a timed loop automatically.

Major features in Show_Random_Post include:

* Add a a Widget to your Theme.
* Show post by Category on a loop, that loads via AJAX.
* Can set time of loading to 10, 30, 60, 120 seconds.
* Can create so much widgets as you want.

== Installation ==

Upload the Show_Random_Post plugin to your blog, Activate it, go to widgets and add.

1, 2, 3: You're done!

== Screenshots ==
